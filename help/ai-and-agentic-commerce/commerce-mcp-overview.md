# Übersicht über Commerce MCP

[!BADGE Bald verfügbar]{type=Informative tooltip="Diese Funktion wird in Kürze verfügbar sein und kann sich ändern."}

Das Commerce MCP für Adobe Commerce unterstützt Entwickler beim Erstellen benutzerdefinierter agentischer Erlebnisse von der Erkennung bis zum Checkout mithilfe von Echtzeit-Commerce-Daten und -Tools. In diesem Video verwendet ein benutzerdefinierter KI-Assistent den Commerce-MCP, um Konversationen in Konversionen zu verwandeln, indem er die Kundenabsichten versteht und relevante Produktempfehlungen bereitstellt.

## Für wen ist dieses Video bestimmt?

* Commerce-Store-Besitzer
* Merchandiser
* Digitale Marketer

## Videoinhalt

* Konversative Produkterkennung mit natürlicher Sprache anstelle von Filtern und Suchbalken
* Kontextorientiertes Sitzungsmanagement zur Speicherung von Warenkorbinhalten und Kundenpräferenzen
* Bidirektionale Warenkorbverwaltung durch Eingabe, ohne Seitenneuladungen oder Bestätigungsmodale
* Checkout innerhalb der Konversation abgeschlossen, ohne wegzunavigieren
* Kontextuelle Anleitungen nach dem Kauf, die speziell für das gekaufte Produkt gelten
