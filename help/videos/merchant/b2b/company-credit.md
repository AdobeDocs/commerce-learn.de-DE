# Verwalten von Firmenguthaben

{{b2b-feature}

Für Unternehmen, die _[!UICONTROL Payment on Account]_aktiviert ist und Firmenkredite zur Verfügung stehen, können sie bis zum gewährten Kreditlimit auf ihrem Konto Einkäufe tätigen. Wenn diese Option aktiviert ist, können Kunden den Status ihrer Unternehmensgutschrift über ihr Konto-Dashboard überprüfen.

Sie können für jedes Unternehmensprofil die folgenden kreditbezogenen Parameter festlegen:

- Kreditwährung
- Kreditbeschränkung
- Erlauben, die Kreditbeschränkung zu überschreiten
- Grund für die Veränderung
