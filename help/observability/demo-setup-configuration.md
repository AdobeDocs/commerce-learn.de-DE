# Einrichtung und Konfiguration

Erfahren Sie, wie Sie mithilfe standardisierter, korrelierter Telemetrie eine End-to-End-Beobachtung in Adobe Commerce und benutzerdefinierten Anwendungen erreichen. In diesem Video wird erläutert, warum Korrelation und Standardisierung wichtig sind. Außerdem werden das Commerce Observability Module und die IO Live Telemetry Library vorgestellt. Erfahren Sie, wie diese Tools eine einheitliche Überwachung, Protokollierung und Verfolgung für einen nahtlosen Betriebsablauf ermöglichen.

## Für wen ist dieses Video bestimmt?

* Adobe Commerce Developer
* Lösungsarchitekt
* Technischer Produkt-Manager

## Videoinhalt

* Standardisierung und Korrelation der Telemetrie für volle Beobachtbarkeit
* Verwenden des Commerce Observability Module für automatisierte Signale
* Erweitern der Überwachung mit der IO Live Telemetry Library für Apps
