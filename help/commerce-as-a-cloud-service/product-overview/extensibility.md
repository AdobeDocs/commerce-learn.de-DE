# Erweitern von Adobe Commerce mit App Builder und API Mesh

Erfahren Sie, wie Adobe Commerce as a Cloud Service die Erweiterbarkeit durch App Builder und API Mesh unterstützt. Erfahren Sie, wie Sie Backend-Systeme integrieren, Workflows automatisieren und benutzerdefinierte Apps erstellen, ohne das Storefront-Erlebnis zu stören.

## Für wen ist dieses Video bestimmt?

* Lösungsarchitekt
* Backend-Entwicklerperson
* Integrationsspezialist

## Videoinhalt

* Erfahren Sie, wie App Builder die Entwicklung benutzerdefinierter Apps und die Workflow-Automatisierung für Adobe Commerce ermöglicht.
* Erfahren Sie, wie API Mesh Daten über mehrere Systeme hinweg über eine einheitliche API-Ebene orchestriert.
