# JavaScript-Modul hinzufügen

{{only-for-on-prem-commerce-cloud}}

Erfahren Sie, wie Sie ein einfaches JavaScript-Modul entwickeln, das die Begrüßung „Hello World“ auf der Storefront anzeigt.

## Für wen ist dieses Video bestimmt?

* Entwickler

## Schritte

1. Erstellen Sie ein Modul.
2. Hinzufügen von `requirejs-config.js` und einer JavaScript-Moduldatei.
3. Erstellen Sie eine Layout-Aktualisierung, die eine Vorlage für das Modul lädt.
4. Erstellen Sie die Vorlagendatei.
5. Registrieren Sie das Modul und testen Sie es auf der Storefront.

## Videoinhalt

* Richten Sie das Modul ein und fordern Sie die JS-Konfiguration an.
* Verkabeln Sie das Skript über eine Layout- und Aktualisierungsvorlage.
* Überprüfen Sie die Begrüßung auf der Seite.
